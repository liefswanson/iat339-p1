Interactive Elements

In-text link

CSS selector:

.text-link

Sample code:

<a href='https://en.wikipedia.org/wiki/Hyperlink' class='text-link'>it's a link!</a>

Rendered element:

it's a link!

Navigation Link

CSS selector:

.button-nav

Sample code:

<a id='nav-to-self' href='#nav-to-self' class='button-nav'>Navigate!</a>

Rendered element:

Navigate!
